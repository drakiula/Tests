Nokia's S40 now also allows W3C Widget-like apps. The interesting thing here is that they will be executed by Nokia's Gecko-based Xpress, which is a proxy browser. So if I understand correctly the apps will run on the server and send their output to the client.

Years ago I heard Opera was working on something similar for Opera Mini, but nothing ever came of it. Good to see that Nokia is continuing this interesting line of thought. But Xpress will have to become a better browser before the S40 app ecosystem will truly grow.

Read more: http://www.developer.nokia.com/Resources/Library/Series_40_web_apps_library/#!index.html
