Datalist behavior

Apparently, in some browsers, the browser's default autocomplete gets added to any results returned by a datalist. Makes sense until you start seeing duplicate items in the list, in which case it gets confusing.

To see it in action, try submitting the form a few times with values from the datalist (Hotdog, Hotdog2, Hotdog3, Hotdog4, French Fries).

Results

  • Chrome 20+: Appends
  • Chrome Mobile: Datalist not supported
  • Firefox 4+: Prepends
  • Opera 10+: Only shows datalist options
  • IE 10: Only shows datalist options
  • Safari: Datalist not supported
  • Android: Datalist not supported
  • iOS: Datalist not supported
