Basic Architecture

If Turbo Mode is enabled, the client makes a request, the request is routed through Yandex's servers. The servers initiate each request over their own networks (sending an X-Forwarded-For header) and compresses them.

Turbo mode is automatically triggered if the connection speed falls below 128 Kbit/s. It will remain on until the speed exceeds 512 Kbit/s. Turbo mode can also be enabled or disabled by the user.

With Turbo mode enabled, mutlimedia is hidden by default.
