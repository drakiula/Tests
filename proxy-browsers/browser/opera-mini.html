Opera Mini

Opera Mini is a proxy browser that focuses on both decreasing bandwidth use as well as enabling lower-powered devices to better experience the web. Pages are requested by their servers, rendered on the server, and then passed to the client in a proprietary format.

Can also be enabled using "Off-Road" mode in Opera mobile.

Basic Architecture

When the client makes a request, the request is routed through Opera's servers. Depending on which client you are using, this may be done over traditional HTTP or via a socket that is kept open during the session (providing a boost in performance). The servers perform the initial DNS resolution. They also initiate each request over their own networks (sending an X-Forwarded-For header). The pages are rendered on the server, and all JS execution takes place on the server.

The page is sent back to the client in a proprietary format using OBML. Whenever something happens that may change the DOM or Render Tree, the client makes a request to the server. The server sends back the updated snapshot of the page for the client to display
