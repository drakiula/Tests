Opera Max

Opera Max is a proxy service (using Opera Turbo) that focuses on decreasing bandwidth for all apps.

Summary

Data Savings Up to 50%
HTTPS Behavior Bypasses proxy
Network Optimization Unclear
Transcoding/Compressing Images Compression
Content compression Compresses all content
X-Forwarded-For Header Yes
Additional Headers HTTP_X_CONTENT_OPTA
Cache-control: no-transform Not sure
Rendering Behavior N/A
Rendering Engine N/A

Basic Architecture

When the client makes a request, the request is routed through Opera's servers. The servers initiate each request over their own networks (sending an X-Forwarded-For header) and compresses them.
